# iris_network

A project intended to build a 2 layered neural network for iris flower recognition


## Installation

```bash
git clone https://github.com/gops26/iris_network.git
cd iris_network
# Install dependencies as needed (for Python, for example:)
# pip install -r requirements.txt
```


## Project Structure

- `src/` - Main source code for image processing, networking, and database management.
- `tests/` - Unit and integration tests.
- `configs/` - Configuration files for different environments/examples.
- `docs/` - Additional documentation and design specifications (if present).
